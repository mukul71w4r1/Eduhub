## 📖 About the Project

EduHub is designed to simplify the learning journey by bringing multiple educational platforms together in one place.

Instead of searching across different websites, students can explore courses, compare learning platforms, and find resources that match their interests and career goals.

The project focuses on creating an intuitive user experience with a clean interface and responsive design.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/mukul71w4r1/Eduhub.git
```

### Go to Project Folder

```bash
cd Eduhub
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Start Backend

```bash
node server.js
```

### Open Frontend

Open `index.html` in your browser.
